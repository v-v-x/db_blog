"Букля" для чтения книг на Маке

2012-02-07 (Last Modified: 2012-02-07)

via macworld.com

Полку программ для чтения EPUB на десктопе прибыло: Bookle для Mac OS X. Программа делает ровно это — отображает EPUB. Относительно корректно и в «родном» интерфейсе Mac. Fixed layout EPUB отображает постранично. Аудио и видео, правда, не воспроизводит. Но с работает. Важно — понимает внедренные шрифты и корректно отображает кириллицу даже без внедренных шрифтов. Позволяет менять шрифт текста — для любителей. Программа продаётся в Mac AppStore за $9.99. С учётом минусов, наверное, могла бы стоить и дешевле.
