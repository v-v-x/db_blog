Исправляем Kindle Previewer в OS X Yosemite

2014-10-31 (Last Modified: 2014-10-31)

После установки OS X Yosemite программа для конвертации epub в mobi Kindle Previewer перестаёт нормально работать. Чтобы как-то починить, устанавливаем Java для OS X, а затем последнюю версию xQuartz. После этого Kindle Previewer так-таки будет генерировать и показывать файлы mobi. Но чтобы увидеть получившийся файл на диске, надо поменять расширение у новообразованной папки (да, она с расширением) с .epub на .mobi. И в этой папке будет нужный mobi-файл.
